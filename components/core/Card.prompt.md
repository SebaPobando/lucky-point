Rounded surface container for menu sections, store products, info panels.

```jsx
<Card variant="plain" interactive pad="md">…</Card>
<Card variant="forest" pad="lg">Programa de puntos</Card>
```

Variants: `plain` (white), `sand` (cream), `dark` (espresso), `forest` (dark green). Use `interactive` for clickable cards (adds hover lift). 20px radius, soft brown-tinted shadow.
