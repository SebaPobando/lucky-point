A single row in the digital carta: name, description, and the dual CLP / Lucky Points price.

```jsx
<MenuItem name="Long Black" clp={3200}
  description="Espresso doble vertido sobre agua caliente." />
<MenuItem name="Mokka Nutella" clp={5500} tag="Favorito" />
```

Composes `PriceTag`. Stack many inside a category section; last item can set `divider={false}`.
